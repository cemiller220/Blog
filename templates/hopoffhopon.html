Hop Off, Hop On

February 7, 2017

        Hop Off, Hop On was the title of my capstone project at The Data Incubator, an 8-week data science bootcamp that I attended this past January and February. The purpose of the project was to create a web application to predict the crowdedness of NYC subway stations and help make transfers more convenient. I used a combination of the subway schedule data and taxi ride data to complete the project.


        The tools I used for this project included many Python libraries, such as Pandas, Numpy, Scikit-learn, and Matplotlib, web development with Flask and HTML, and interactive data visualization with Javascript (specifically nvd3 and dc.js).


        To see the completed project, I encourage you to check out the final website hop-off-hop-on.herokuapp.com. There you can read more details about how I did my analysis and use the application to look up your desired subway route.
